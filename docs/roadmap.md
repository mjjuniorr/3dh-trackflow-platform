# Roadmap

## Fase atual: aplicativo Android

- Criar aplicativo Android do entregador para publicar localizacao em tempo real.
- Usar cadastro mobile para criar ou reativar o entregador pelo `device_id`.
- Enviar `device_id`, latitude, longitude, velocidade, heading, bateria, acuracia e timestamp.
- Validar integracao ponta a ponta com Kafka, backend, dashboard e link publico.

## Tarefas futuras

- Definir tipo de veiculo por entregador para escolher icone de moto, carro ou outro modal.
- Criar historico de rotas por entregador para auditoria e replay.
